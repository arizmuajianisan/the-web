# My Personal Website

This is my personal website built with Astro.js. It features a modern, responsive design and is optimized for performance and SEO.

## Features

- Modern, responsive design
- SEO optimized
- Dark mode support
- Fast loading times
- Easy to navigate
- Easy to update
- Easy to deploy
- Easy to maintain


## Installation

```bash
npm install
```

## Running the Project

```bash
npm wrangler dev
```

## Building the Project

```bash
npx astro build
```

## Deploying to Cloudflare Workers

```bash
npx wrangler deploy
```
